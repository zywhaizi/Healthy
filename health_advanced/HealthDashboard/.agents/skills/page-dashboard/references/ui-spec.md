# page-dashboard · UI 规格

## 卡片布局（自上而下）

```
┌─────────────────────────────┐
│  步数卡片（全宽）             │
│  ┌────────┐                 │
│  │ 进度环  │  今日步数        │
│  │  Ring  │  目标 / 卡路里   │
│  └────────┘                 │
└─────────────────────────────┘

┌──────────────┐ ┌────────────┐
│  喝水卡片    │ │  心情卡片  │
│  HDWaterView │ │  最新心情  │
└──────────────┘ └────────────┘

┌─────────────────────────────┐
│  睡眠卡片（全宽）             │
│  HDSleepBarView（7天柱状）   │
└─────────────────────────────┘
```

## 间距规格

- 页面左右边距：16pt
- 卡片间垂直间距：12pt
- 两列卡片水平间距：12pt
- 卡片内边距：16pt
- 卡片圆角：12pt

## 进度环规格（HDRingProgressView）

- 外径：建议 120pt
- 线宽：12pt
- 背景圆颜色：`UIColor.systemGray5Color`
- 进度颜色：根据完成度动态变色（参考 design-system Skill）
- 动画时长：0.6s，easeInOut

## 睡眠柱状图规格（HDSleepBarView）

- 柱子数量：7（固定）
- 柱宽：(卡片宽 - 32 - 间距*6) / 7
- 最大高度：对应8小时
- 推荐睡眠范围：7-9小时（绿色高亮）
- X轴标签：「周一」～「周日」
